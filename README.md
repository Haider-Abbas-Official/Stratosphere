# Stratosphere

Modern self-hosted local cloud storage application built with **Qt**, **Node.js**, **JavaScript**, and **HTML/CSS**, enabling secure file sharing and management across devices connected to the same local network.

---

## ✨ Features

- ☁️ Self-hosted local cloud storage
- 🌐 Browser-based file management interface
- 📤 File upload and download
- 📁 Browse and organize files from any connected device
- 🖥️ Qt desktop server application
- 📡 LAN-based access using the server's IP address and port
- ⚡ Lightweight Node.js backend

---

## 🛠 Technologies

- C++
- Qt Widgets
- Node.js
- JavaScript
- HTML5
